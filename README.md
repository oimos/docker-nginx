###Check Config
`docker-compose config`

###Show Conatiner List
`docker-compose ps`

###Show Container Using Port
`sudo lsof -i:<PORT>`

###Kill Port
`sudo kill <PID>`

###Connect to DB
`docker exec -it <Container Name>`

After Connect to mysql in Container

`mysql -h127.0.0.1 -u<User Name> -p<Password>`
